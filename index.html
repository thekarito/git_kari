Iniciando con Git
Guía rápida para principiantes

1. Como comenzar?

  • Debemos crear nuestro repositorio remoto git a través de nuestra cuenta Github
  • Para crearlo simplemente debes indicar el nombre y el tipo de acceso (Publico/privado)
  • Seguidamente nos mostrara una guía con las 4 opciones principales para iniciar nuestro repositorio, como somos principiantes, elegimos la opción que nos indica: "Crear repositorio nuevo en la linea de comandos"
  • Luego, nos ubicamos en la terminal de comandos provista en nuestro entorno de desarrollo o en la consola de comandos propia de nuestro sistema operativo Linux

2. Como crear o iniciar nuestro primer repositorio local con Git?

Debemos ubicarnos en la carpeta del proyecto donde iniciaremos nuestro repositorio Git 
cd ruta/directorio
Ejecuta el siguiente comando para comenzar 
git init  // Repositorio Git vació, inicializado en /.git
Ahora ejecuta el comando 
git status //Te permitirá visualizar la lista de archivos actuales, archivos por agregar a git, cambios..

3. Como agregar los archivos y directorios de nuestros proyecto al repositorio local Git?

Ejecuta el comando 
git add *   //dicho comando permitira agregar los archivos incluidos dentro de nuestro proyecto

Luego ejecuta 
git status //Te permitira visualizar el estado actual de tu proyecto una vez ejecutado el comando anterior

4. Crear nuestro primer commit?

Ejecuta el comando 
git commit -m "Mensaje que describa los cambios"   //En git todos los cambios deben ser comiteados antes de ser almacenados en nuestro repositorio

Ahora bien, para observar la lista de los commit y conocer los detalles de cada uno, ejecuta el comando 
git log

5. Como enviar nuestro repositorio local al servidor Github?

Debemos añadir nuestro repositorio remoto, ejecutando lo siguiente: 
git remote add origin https://github.com/thekarito/git_kari.git  //Recuerde ubicarse en la linea  git remote add origin https: ...., proporcionada por la guía de configuración de su repositorio en Github

6. Finalmente, como empujar o subir nuestros cambios locales a nuestro repositorio remoto Github?

Para subir los cambios, ejecuta lo siguiente: 
git push -u origin master  //Tengamos en cuenta que nuestro repositorio remoto es origin y el nombre de referencia a nuestro repositorio local predeterminada es master

No olvide indicar el usuario y contraseña de su cuenta de Github, para que la operación sea exitosa

Ahora si, podemos decir que hemos creado nuestro primer repositorio Github :-)
