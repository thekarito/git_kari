Iniciando con Git
Comandos Basicos

1. Como comenzar?

  • Debemos crear nuestro repositorio git en nuestra cuenta Github o bitbucket
  • Simplemente debemos indicar el nombre y el tipo de acceso (Publico/privado)
  • Seguidamente nos mostrara una guia con las 4 opciones principales para iniciar nuestro repositorio, como somos principiantes, elegimos la opción que nos indica: "Crear repositorio nuevo en la linea de comandos"

1. Como crear o iniciar nuestro primer repositorio con Git?

Debemos ubicarnos en la carpeta del proyecto donde iniciaremos nuestro repositorio Git 
cd ubicacion_del_directorio/
Ejecuta el siguiente comando para comenzar 
git init
Ahora ejecutaremos el comando 
git status
, para observar el estado actual de nuestro proyecto Dicho comando nos mostrara la lista de archivos que posee nuestro proyecto, indicando además que los mismos no han sido agregados aun
